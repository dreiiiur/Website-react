# Marketing Services Landing Page

A modern, responsive landing page for marketing services built with React and Tailwind CSS.

## Features

- Hero section with call-to-action
- Services showcase
- About section
- Contact form
- Fully responsive design

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
```

## Technologies Used

- React 18
- Vite
- Tailwind CSS
- ESLint
